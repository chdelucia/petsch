## 2026-04-15 - [Accessible Action Buttons]
**Learning:** When adding interactive elements like "Remove" buttons in lists, they must have clear focus states and semantic meaning. Using Tailwind's `focus-visible` ensures that keyboard users can identify the active element without cluttering the UI for mouse users.
**Action:** Always apply `focus-visible` rings to buttons and ensure they use the primary accent color (e.g., indigo-500) for consistency with the design system.
