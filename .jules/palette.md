## 2025-05-15 - [Accessible Dropdowns and Keyboard Navigation]
**Learning:** Collapsible UI elements like dropdown filters require explicit ARIA state signaling (aria-expanded) and keyboard escape hatches (Escape key) to meet accessibility standards and user expectations for interactive micro-UX.
**Action:** Always verify that components toggling visibility of content implement aria-expanded, aria-controls, and respond to the Escape key for closure.
